ApplyFlow is a modern and responsive Job Application Manager built with React and Tailwind CSS.
It helps users efficiently track, manage, and organize their job applications in one place.

This project demonstrates practical implementation of CRUD operations, state management, and local storage persistence in a real-world application.

**рџЊђ Live Demo:** [https://applyflow-react.vercel.app/](https://applyflow-react.vercel.app/)

## вњЁ Features

- **Full CRUD Operations:** Create, view, update, and delete job applications with ease.
- **In-place Editing:** Edit job details directly within the job card without navigating away.
- **Real-time Status Management:** Instantly update application status (Pending, Interview, Rejected).
- **Smart Filtering:** Quickly filter applications based on their current status.
- **Auto-Calculating Summary:** Dynamic summary dashboard (Total, Pending, Interview, Rejected).
- **Persistent Storage:** Uses **Local Storage** to save applications even after closing the browser.
- **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).

## рџљЂ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash

   ```
2. Navigate to the project folder:
